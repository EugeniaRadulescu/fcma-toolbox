Demonstration

An early preview follows, extremely limited in its scope and in resources that can be devoted to it at this time, but which will be rapidly evolving in the coming months.
Please upload four (4) files:

  • Data: an anonymized 3D + time FMRI volume in NiFTI .nii.gz format
  • Mask: a 3D binary spatial mask, same space dimensions as data (.nii.gz format)
  • FCMA File: an .fcma file generated via the command line or via the FCMA File Generator.
  • Blockfile: a .txt file containing a description of experiment trials; see the FCMA paramater reference for the format.

    • Your files will be uploaded and analyzed on a small cluster of computers (currently 8 max). The files are automatically deleted after the analysis ends, but they can be deleted manually from the list below at any time.


The demonstration is currently off-line.

Demo Notes

  • The maximum file size is 100 MB.
  • File extensions recognized: (.nii.gz for data & masks, .txt for blockfile and .fcma for the .fcma file).
  • You can drag & drop files from your desktop on this webpage with Google Chrome, Mozilla Firefox and Apple Safari.
